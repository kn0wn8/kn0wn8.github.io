本博客的诞生

GItPage + Hexo + Dynadot域名 + Netlify CDN

简易的静态博客解决方案

Part1: GitPage + Hexo 搭建页面

快速上手视频教程：https://www.bilibili.com/video/BV1Yb411a7ty/?spm_id_from=333.788.recommend_more_video.-1&vd_source=f3dc3a3e679a8cdb0b563bc2c1ab0eeb
git -> npm -> hexo -> git_repo
以下内容缺一不可！

tips:
npm 换源命令：npm config set registry https://registry.npmmirror.com
换回来： npm config set registry https://registry.npmjs.org
检测源： npm config get registry

加载hexo部署模块: npm install hexo-deployer-git –save
hexo init
hexo clean && hexo g && hexo d // 部署一条龙

github repo 格式为 username.github.io
vim _config.yml
type:git
repo:xxxxx

扩展：在ECS中部署Hexo
https://blog.csdn.net/GsL9D1P04F6x2ts3C9/article/details/81024330

Part2: Dynatot域名 + Netlify CDN

dynatot.com/zh/ 域名有中文界面、支持支付宝付款、免实名
Netlify CDN 免费版每月前三个网站共100个G流量，很划算，降低网络波动，对图片资源的提速效果明显

方法：
echo “你买的域名“ >> blog/source/CNAME
hexo clean && hexo g && hexo d
将Netlify配置的域名做为 Dynatot的DNS CNAME 解析
可以再添加一个 www.你买的域名.xxx CNAME 到Netlify会自动跳转到主域名
